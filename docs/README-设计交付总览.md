## 🎨 核心设计决策总结

### 设计风格
**Editorial Magazine Style** - 以内容为王的杂志排版风格

**选择理由:**
1. 纪传体本质是文学化生命故事，Editorial天然适合长篇叙事
2. 银发群体熟悉纸媒阅读习惯（杂志/书籍）
3. 温暖米白+琥珀金传递"有温度的数字生命遗产"
4. 央视品质感要求克制、精致的视觉语言

### 色彩策略
```
主文本:   #1A1A1A (近黑，AAA级 15.8:1)
页面底色: #FEFEFE (温暖白)
品牌色:   #D4A574 (琥珀金，象征时光传承)
```

**85-10-5分配:**
- 85% 中性色（文本+背景）
- 10% 琥珀金（CTA+高亮+边框）
- 5% 真彩图像（镜像+照片）

### 字体系统
```
标题:  Noto Serif SC (中文优化衬线)
正文:  Georgia (易读衬线)
UI:    Inter (现代无衬线)
```

**字号层级（桌面端）:**
- Display: 64-80px（传记封面）
- Headline: 40-48px（篇章标题）
- Body: 18-20px（正文，银发群体适配）
- Metadata: 14px（时间戳、标签）

### 间距系统
**8pt网格:** 8px, 16px, 24px, 32px, 48px, 64px, 96px, 128px

**关键原则:**
- 段落间距≥24px（呼吸感）
- 章节分隔≥96px（明确断点）
- 按钮触摸目标≥56px（银发群体）

### Editorial经典元素
1. **Drop Cap（首字下沉）:** 72px琥珀金大字，章节开篇
2. **Pull Quote（突出引用）:** 28-32px斜体，左侧4px琥珀金边框
3. **图片说明:** 16px斜体灰色，紧贴图片
4. **居中文章列:** 650px最大宽度（60-75字符）

## 🎯 关键界面设计

### 三区联动访谈界面（核心创新）

```
┌──────────────────────────────────────────┐
│  顶栏: 进度指示 + 保存草稿 + 退出        │
├───────────┬──────────────┬───────────────┤
│ AI记者区  │  镜像引擎区  │  故事锚点区   │
│ (25%宽)   │  (50%宽)     │  (25%宽)      │
│           │              │               │
│ 记者头像  │  镜像画布    │  时代素材     │
│ 对话气泡  │  (1:1比例)   │  (收音机/粮票)│
│ 追问选项  │  年代选择器  │  事件时间线   │
│           │              │               │
└───────────┴──────────────┴───────────────┘
│  底栏: 用户输入区（文字/语音）           │
└──────────────────────────────────────────┘
```

**交互流程:**
1. AI记者提出"记忆钥匙"问题（感官切入）
2. 用户文字/语音回答
3. 镜像引擎实时生成对应年代形象
4. 右侧时间线自动添加事件锚点
5. AI共情反馈 + 追问选项
6. 循环直到篇章完成

### 传记展示页面（Editorial排版）

```
Hero封面 (70vh，全屏背景)
  ↓
居中文章列 (650px)
  ├─ Drop Cap开篇
  ├─ 正文段落（24-32px间距）
  ├─ Pull Quote（每3-5段）
  ├─ 插图 + 说明
  └─ 继续正文
  ↓
分享导出区
```

### 桌面端 (≥1024px)
- 三区并列布局（25%-50%-25%）
- 传记文章列650px居中

- AI记者收起为抽屉（点击展开）
- 镜像+锚点（60%-40%）

### 手机端 (<768px)
- Tab切换三区（对话 | 镜像 | 时间线）
- 固定底栏输入
- 字号保持18px（阅读优先）
- 按钮全宽，触摸目标56px
