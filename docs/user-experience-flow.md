**版本:** 1.0 | **日期:** 2025-10-31

## 1. 核心用户旅程

### 1.1 首次访问流程（新用户）

```
开始
  ↓
[欢迎引导页] (/welcome)
  - Hero区：视觉冲击 + 核心价值主张
  - 品质背书：央视团队、高端案例
  - 产品包对比：基础/进阶/尊享
  ↓
  [注册/登录] (模态框)
  - 手机号 + 验证码（简化流程）
  - 可选：微信/支付宝快捷登录
  ↓
[引导教程] (首次登录Onboarding)
  步骤1: "您的生命故事值得被珍藏"（情感建立）
  步骤2: "AI记者将温暖陪伴您的回忆之旅"（功能介绍）
  步骤3: "随时保存，慢慢讲述"（消除压力）
  ↓
[人生篇章选择] (/chapters)
  结束
```

**关键体验点:**
- **情感建立:** 欢迎页用温暖文案建立信任，避免冰冷注册表单
- **降低门槛:** 注册流程≤3步，支持社交账号快捷登录
- **消除焦虑:** Onboarding强调"无需一次完成"，减轻银发群体压力

```
  ↓
  用户点击篇章（如"童年故里"）
  ↓
[记忆回溯访谈界面] (/interview/childhood)
  
  ┌────────────────────────────────────────┐
  │  顶栏: "童年故里 - 第3/12问" + [保存草稿] + [退出] │
  ├──────────┬──────────┬──────────────────┤
  │          │          │                  │
  │ 【左侧】 │ 【中央】 │    【右侧】      │

  │          │          │                  │
  │ ① 记者头像│ ④ 镜像画布│ ⑥ 时代素材      │
  │   点头动作│   生成中... │   [收音机图标]  │
  │          │          │   [粮票]        │
  │ ② 对话气泡│ ⑤ 年代标签│   [歌曲播放]    │
  │  "闭上眼睛│   1980年代│                  │
  │   还记得老│          │ ⑦ 事件时间线    │
  │   家门口那│          │   ● 1975 出生   │
  │   条路吗？"│          │   ● 1980 上学   │
  │          │          │   ○ ...         │
  │ ③ 追问选项│          │                  │
  │  [路的材质]│          │                  │
  │  [雨后气味]│          │                  │
  └──────────┴──────────┴──────────────────┘
  │  ⑧ 底部输入栏: "青石板，雨后有泥土混着青草的味道..."  │
  │     [文字输入]  [语音输入🎤]  [发送→]            │
  └────────────────────────────────────────┘
  
  交互循环:
  1. AI记者提出"记忆钥匙"问题（感官切入）
  2. 用户文字/语音回答
  3. 系统解析关键词（时间、地点、物件）
  4. 镜像引擎自动生成对应年代用户形象
  5. 右侧时间线添加新事件锚点
  6. AI记者共情反馈："一条泥土路，雨后的味道，这个画面太生动了"
  7. 提供追问选项（用户选择 or 继续自由讲述）
  8. 重复步骤1-7，直到篇章完成（12-15问）
  ↓
  [篇章完成确认]
  - 显示本篇章时间线回顾
  - 提示："您已完成「童年故里」，休息一下，或继续下一篇章？"
  - [返回篇章选择] / [继续下一篇章]
```

**关键体验点:**
- **沉浸式对话:** 三区联动，视觉+听觉+交互，避免单调问答
- **即时反馈:** 每次回答后AI给出共情反馈，增强情感连接
- **控制权在用户:** 追问选项而非强制，用户可选择深挖或跳过
- **自动保存:** 每问自动保存，用户可随时退出

## 2. 关键用户场景

### 场景A: 70岁退休老教授，首次使用

**用户画像:** 王教授，70岁，退休大学教师，对新技术谨慎但好奇，希望留下家族记忆

**旅程:**
1. **发现阶段:** 子女推荐，访问欢迎页
   - 被"央视团队"背书吸引，建立初步信任
   - 看到尊享包样片，感受到品质感

2. **犹豫阶段:** 担心操作复杂
   - Onboarding教程消除焦虑："慢慢讲述，随时保存"
   - 看到大字号界面，感觉友好

3. **尝试阶段:** 进入"童年故里"访谈
   - AI记者温暖提问："闭上眼睛，还记得老家门口那条路吗？"
   - 王教授被唤起回忆，语音输入（不擅长打字）
   - 镜像引擎生成1950年代形象，惊喜："这就是我小时候的样子！"

4. **沉浸阶段:** 完成3个篇章
   - 右侧时间线逐渐丰满，成就感提升
   - AI记者共情反馈让他感到被理解
   - 每次退出前自动保存，消除担忧

5. **完成阶段:** 生成传记
   - 选择"余秋雨风格"（文化哲思契合教师身份）
   - 阅读传记时多次落泪，点击"分享链接"发给子女
   - 子女回复："爸，这就是您的一生啊"，王教授决定升级尊享包

**关键触点:**
- ✅ 央视背书建立信任（减少老年人防御心理）
- ✅ 语音输入降低操作门槛
- ✅ 镜像引擎的"惊喜时刻"成为口碑传播点
- ✅ 自动保存消除"半途而废"焦虑

---

### 场景B: 55岁企业家，时间有限

**用户画像:** 李总，55岁，成功企业家，日程繁忙，希望快速完成传记

**旅程:**
1. **快速决策:** 看重"央视导演团队"，直接选择尊享包
2. **碎片化访谈:** 利用出差飞机/酒店时间，分多次完成访谈
   - 感激"自动保存"功能，无需担心中断
3. **高效选择:** 文风选择直接"刘慈欣风格"（宏大叙事匹配企业家气质）
4. **商务用途:** 导出PDF后用于企业内刊，展示创始人故事

**关键触点:**
- ✅ 碎片化完成机制（支持多设备同步）
- ✅ 快速路径（跳过教程，直奔功能）
- ✅ 商务化输出（PDF高质量排版）

## 3. 异常流程与容错设计

### 3.1 访谈中断恢复

```
用户中途退出访谈（如接电话）
  ↓
系统自动保存当前进度
  ↓
用户下次登录
  ↓
[弹窗提示]
  "欢迎回来！您上次在「童年故里」进行到第5问
   [继续访谈] [重新开始此篇章]"
  ↓
  用户点击"继续访谈"
  ↓
  直接跳转到第5问，AI记者简要回顾：
  "刚才您提到老家门口的石板路，我们继续聊聊..."
```

### 3.2 镜像生成失败

```
用户上传照片 → 镜像引擎生成
  ↓
  [生成失败]（如照片不清晰、服务器超时）
  ↓
  [友好提示]
  "抱歉，这张照片生成遇到了小问题
   建议：
   - 重新上传更清晰的照片
   - 或跳过镜像，先完成访谈
   
   [重新上传] [暂时跳过]"
  ↓
  用户选择"暂时跳过"
  ↓
  访谈继续，右侧时间线正常工作
  （镜像非必需，不阻塞核心流程）
```

### 3.3 风格预览加载慢

```
用户点击"预览我的故事"
  ↓
  [骨架屏] + 文案："AI正在渲染风格..."
  ↓
  超时15秒仍未加载
  ↓
  [降级方案]
  - 显示通用示例段落（预置）
  - 提示："网络较慢，显示示例仅供参考"
  - 用户可直接选择风格，无需等待
```

## 4. 情感化设计细节

### 4.1 微文案（Microcopy）

| 场景 | 冰冷写法 ❌ | 温暖写法 ✅ |
|------|-------------|-------------|
| 注册按钮 | "提交" | "开启我的生命故事" |
| 访谈保存 | "保存成功" | "您的记忆已珍藏，随时可继续" |
| 篇章完成 | "完成" | "这一章很精彩，休息一下吧" |
| 传记生成中 | "处理中..." | "AI正在以[作家]的笔触，细细描绘您的故事..." |
| 分享成功 | "链接已复制" | "链接已生成，与家人分享这份传承" |
| 空状态 | "暂无内容" | "您的故事即将在这里绽放" |

### 4.2 动效情感化

**欢迎动画:**
- 页面加载时，标题从模糊到清晰淡入（象征记忆浮现）
- 时长600ms，避免过快

**篇章卡片悬停:**
- 卡片微抬升（-4px）+ 阴影加深
- 配合微妙的暖色光晕（琥珀金5% opacity）
- 传递"邀请触碰"的温暖感

**时间线节点添加:**
- 新事件出现时，从右侧滑入 + 琥珀金高亮闪烁1次
- 象征"记忆锚定"的仪式感

### 4.3 声音设计（可选增强）

| 场景 | 声音 | 目的 |
|------|------|------|
| 篇章完成 | 轻柔的铃声（类似书页翻过） | 仪式感，成就反馈 |
| AI记者提问 | 柔和的"咚"（非机械音） | 引起注意，不刺耳 |
| 传记生成完成 | 舒缓的弦乐（3秒淡出） | 庄重感，情感高潮 |

**注意:** 声音默认关闭，用户可在设置中开启（银发群体可能不适应突然声音）

## 5. 无障碍与包容性设计

### 5.1 视觉无障碍

- **高对比模式:** 设置中可切换，文本对比度提升至21:1
- **字号调节:** 三档可选（默认18px / 大20px / 特大24px）
- **颜色盲友好:** 琥珀金与墨绿避开红绿色盲区，状态不仅用色彩区分

### 5.2 操作无障碍

- **键盘导航:** 所有操作支持Tab键遍历 + Enter确认
- **语音输入优先:** 银发群体打字慢，语音输入按钮放在显眼位置
- **撤销功能:** 所有关键操作（删除、退出）提供二次确认 + 撤销

### 5.3 认知友好

- **进度可见:** 顶部固定显示"第X/12问"，降低迷失感
- **上下文提示:** AI记者回顾前文，降低记忆负担
- **一次一问:** 避免一屏多个问题，减少认知负荷

## 6. 成功指标（UX Metrics）

| 指标 | 目标值 | 测量方法 |
|------|--------|----------|
| **首次访问完成率** | ≥60% | 从欢迎页到完成第1个篇章的用户占比 |
| **篇章平均完成时长** | 15-25分钟 | 单篇章从开始到完成的平均时间 |
| **中途放弃率** | ≤30% | 开始访谈但未完成任何篇章的用户占比 |
| **镜像生成成功率** | ≥90% | 镜像引擎生成成功次数 / 总请求次数 |
| **传记分享率** | ≥40% | 生成传记后使用分享功能的用户占比 |
| **NPS（净推荐值）** | ≥50 | 用户愿意推荐给他人的意愿评分 |
