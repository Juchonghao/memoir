## 2. 应用结构

**类型:** 单页应用(SPA)

**理由:** 
- 核心功能是连续的访谈流程，需保持状态连贯
- 三区联动界面需要实时交互和状态同步
- 沉浸式体验要求减少页面跳转打断
- 记忆数据需在多个视图间共享

## 4. 内容分析

**信息密度:** 中等
- 访谈界面：实时交互为主，文本密度适中
- 传记展示：长文本内容，需优化阅读体验

**内容平衡:**
- 文本: 60% (访谈对话、传记正文)
- 图像: 30% (镜像引擎生成图、时代素材、用户照片)
- 交互组件: 10% (按钮、选择器、输入框)

**内容类型:** 叙事为主+数据辅助
- 核心是用户生命故事的叙事化表达
- 时间轴、进度等数据可视化为辅

## 5. 设计关键约束

**必须遵循（基于Editorial风格指南）:**

✅ **排版:**
- 传记正文：居中文章列，650-750px最大宽度
- 正文字号：18-20px（适配银发群体阅读）
- 行高：1.6-1.8（舒适阅读）
- 段落间距：24-32px（呼吸感）

✅ **色彩:**
- 主体：85%中性色（#1A1A1A文本 + #FEFEFE背景）
- 点缀：10%琥珀金#D4A574（象征时光与传承）
- 图像：5%保留真彩（镜像引擎、用户照片）

✅ **字体:**
- 标题：Noto Serif SC（中文场景优化）/ Playfair Display（英文备选）
- 正文：Georgia / 思源宋体
- UI元素：Inter（导航、按钮、元数据）

✅ **组件:**
- Drop Cap：章节开篇首字下沉
- Pull Quote：关键记忆片段的突出引用
- 图片说明：Italic 16px，灰色

❌ **禁止:**
- 多彩背景（仅用白色/米白色）
- 圆角 >8px（保持Editorial锐度）
- 过度动效（避免干扰阅读）
- 装饰性渐变

## 6. 特殊考虑：银发群体适配

**视觉适配:**
- 字号整体提升20%（最小18px）
- 对比度≥7:1（AAA级）
- 关键操作按钮≥56px高度

**交互适配:**
- 避免复杂手势（简化为点击、滑动）
- 清晰的状态反馈（操作后明确提示）
- 容错机制（撤销功能、保存草稿）

**情感设计:**
- 温暖色调（米白底色+琥珀金点缀）
- 亲切文案（避免技术术语）
- 尊重感（传记展示强调"传承价值"）
