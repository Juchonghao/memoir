## 🐛 常见问题

### 问题1：端口已被占用
**解决**：
```bash
# 查看占用端口的进程
lsof -ti:5173

# 杀死进程
kill -9 <进程ID>

# 或使用其他端口
npm run dev -- --port 3000
```

### 问题2：依赖未安装
**解决**：
```bash
npm install
```

### 问题3：页面无法访问
**检查**：
1. 确认开发服务器正在运行
2. 检查终端是否有错误信息
3. 确认浏览器地址栏输入正确

### 问题4：热更新不工作
**解决**：
1. 保存文件后稍等片刻
2. 检查浏览器控制台是否有错误
3. 尝试手动刷新页面（Cmd+R 或 Ctrl+R）
